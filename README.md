mac-setup
=========

One-liner:

```
(mkdir -p /tmp/dots && cd /tmp/dots && curl -L https://github.com/lukeaskew/mac-setup/archive/master.tar.gz | tar zx --strip 1 && sh ./install.sh)
```

Setup script for new Mac development machine

- Installs Homebrew
- Installs apps
- Installs binaries
- Sets OS defaults
- Configures sublime
- Sets up Apache
- Sets wallpaper

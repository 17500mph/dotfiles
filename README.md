mac-setup
=========

One-liner:

```
(mkdir -p /tmp/osx-setup && cd /tmp/osx-setup && curl -L https://github.com/lukeaskew/osx-setup/archive/master.tar.gz | tar zx --strip 1 && sh ./install.sh)
```

Setup script for new Mac development machine

- Installs Homebrew
- Installs apps
- Installs binaries
- Sets OS defaults
